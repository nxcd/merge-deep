## Instalation

Simply run

**NPM:**
```bash
$ npm install @nxcd/tardis
```

**Yarn:**
```bash
$ yarn add @nxcd/tardis
```

## Usage

### Javascript:

```js
const mergeDeep = require('@nxcd/merge-deep')

const target = { a: 'a' }
const source = { b: 'b' }

mergeDeep(target, source) // { a: a, b: b }

```
